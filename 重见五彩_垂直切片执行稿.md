# 《重见五彩》垂直切片执行稿

> 版本：v0.1  
> 日期：2026-06-16  
> 用途：指导第一版 10-15 分钟灰盒原型与后续 20-35 分钟参赛切片制作。  
> 当前目标：先做一个玩家能独立玩完的小闭环，而不是完整游戏。

## 1. 切片定位

本切片只验证一个核心体验：

> 玩家从一次失败中理解工艺问题，调整参数，再次开窑并看到五彩重现。

切片不是完整章节，也不是完整经营模拟。它只保留“任务-试验-失败-复盘-调整-成功-档案”的主闭环。

## 2. 设计假设

- 平台：优先网页端灰盒原型。
- 操作：鼠标点击为主。
- 美术：第一版可全部使用占位图或纯文字。
- 系统：先做参数选择与结果判定，不做复杂小游戏。
- 角色：先保留 3 个功能角色。
- 时长：灰盒版 10-15 分钟；参赛扩展版 20-35 分钟。

## 3. 核心体验目标

玩家玩完后应能说出：

1. 醴陵釉下五彩不是普通填色，而是釉下彩绘、烧后呈色、开窑揭晓的工艺结果。
2. 三烧改二烧不是简单少烧一次，而是要解决长石釉、罩釉、升温和有机物排出的问题。
3. 失败不是随机惩罚，而是找到下一步改进方向的线索。
4. 抢救复兴是团队协作，不是某个主角单独发明。

## 4. 角色配置

第一版只保留 3 个角色，避免体量膨胀。

| 角色 | 功能 | 是否需要立绘 |
| --- | --- | --- |
| 玩家 / 工坊组织者 | 做选择、安排试验、推动流程 | 不需要 |
| 老艺人 | 说明旧三烧制、判断“像不像” | 第一版可无立绘 |
| 青年技术员 | 提供试验记录、解释失败原因 | 第一版可无立绘 |

配釉师、烧窑师、彩绘师暂时不作为独立角色出现，其功能先合并给青年技术员和老艺人。等切片跑通后再拆分。

## 5. 10-15 分钟灰盒流程

### 0-1 分钟：开始

玩家看到标题和一句话说明。

画面内容：

- 标题：《重见五彩》
- 副标题：在 1950 年代的醴陵，抢救濒临断代的釉下五彩技艺。
- 按钮：开始

玩家操作：

- 点击“开始”。

完成条件：

- 进入工坊界面。

### 1-3 分钟：工坊任务

玩家进入工坊，看到当前任务。

剧情要点：

- 工坊接到恢复釉下五彩的任务。
- 手头只有残片、旧谱和老艺人口述。
- 老艺人提醒：知道步骤不等于能烧成。

玩家操作：

- 阅读短对白。
- 点击“开始第一次试烧”。

完成条件：

- 进入第一次制作参数界面。

### 3-5 分钟：第一次试烧

第一次试烧使用旧流程，不让玩家做太多选择。

默认参数：

- 釉料：旧石灰青釉
- 罩釉厚度：中
- 升温节奏：中
- 氧化气氛：普通

玩家操作：

- 点击“入窑烧成”。

结果：

- 作品基本成形，但颜色发灰、发暗。
- 说明旧工艺能恢复，但呈色和效率存在问题。

完成条件：

- 进入第一次结果界面。

### 5-7 分钟：问题暴露

玩家查看结果和角色反馈。

反馈内容：

- 老艺人：形和线条有些像了，但颜色不够明亮。
- 青年技术员：旧釉底色偏青偏暗，可能影响釉下色呈现。
- 系统提示：解锁“长石釉试验”。

玩家操作：

- 点击“尝试改进工艺”。

完成条件：

- 进入二烧试验参数界面。

### 7-10 分钟：二烧第一次试验

玩家第一次主动选择工艺参数。

可选参数：

- 釉料：旧石灰青釉 / 长石釉
- 罩釉厚度：薄 / 中 / 厚
- 升温节奏：慢 / 中 / 快
- 氧化气氛：不足 / 普通 / 充分

建议默认选择一个会失败的组合：

- 釉料：长石釉
- 罩釉厚度：厚
- 升温节奏：快
- 氧化气氛：普通

结果：

- 触发针孔、气泡或黑壳。

完成条件：

- 进入失败结果界面。

### 10-12 分钟：失败复盘

玩家查看失败原因。

复盘内容：

- 罩釉过厚会阻碍排气。
- 升温太快会缩短有机物排出窗口。
- 二烧制省去独立低温焙烧后，对罩釉和烧成控制更敏感。

角色反馈：

- 青年技术员：这不是少烧一次就能成功，必须让墨线等有机物在釉层封闭前排出去。
- 老艺人：急不得，开窑前没人能保证结果。

玩家操作：

- 点击“调整方案再试一次”。

完成条件：

- 返回参数界面，并显示改进建议。

### 12-14 分钟：第二次试验

玩家根据建议选择更合理方案。

推荐成功组合：

- 釉料：长石釉
- 罩釉厚度：薄
- 升温节奏：慢
- 氧化气氛：充分

结果：

- 作品成功。
- 底色更洁净。
- 色彩更明亮。
- 无明显黑壳、针孔、气泡。

完成条件：

- 进入成功结果界面。

### 14-15 分钟：展柜与档案

玩家看到作品进入展柜。

解锁档案：

- 档案：三烧改二烧
- 档案：长石釉
- 档案：失败复盘记录

结尾文案：

> 这不是一个人的发明，而是一代人把快要断掉的知识重新变成可以传下去的能力。

玩家操作：

- 点击“结束原型”或“重新开始”。

完成条件：

- 完成灰盒闭环。

## 6. 20-35 分钟参赛扩展版

等 10-15 分钟灰盒跑通后，再扩展为参赛版。

| 时间 | 内容 | 扩展点 |
| --- | --- | --- |
| 0-3 分钟 | 开场与工坊困境 | 增加残片、旧谱、角色入场 |
| 3-8 分钟 | 三烧制教学 | 增加简单双勾分水演示 |
| 8-12 分钟 | 第一次开窑 | 增加选瓷和瑕疵识别 |
| 12-18 分钟 | 长石釉试验 | 增加试片墙 |
| 18-25 分钟 | 二烧失败与复盘 | 增加失败档案 |
| 25-31 分钟 | 二烧成功 | 增加开窑演出 |
| 31-35 分钟 | 展柜与史实档案 | 增加真实历史说明 |

## 7. 灰盒对白草案

### 开场

老艺人：

> 釉下五彩不是知道几道工序就能烧出来。线、色、釉、火，哪一步差了，开窑时都会告诉你。

青年技术员：

> 我们先把每一次试烧都记下来。失败不是白费，它会告诉我们下一次该改哪里。

### 第一次发灰

老艺人：

> 形有了，味道还不够。这个颜色，像蒙了一层灰。

青年技术员：

> 旧釉底色偏青偏暗，可能压住了釉下色。我们可以试试更白、更透的长石釉。

### 二烧第一次失败

青年技术员：

> 这批有针孔和黑壳。问题不在画面，而在罩釉和烧成。

老艺人：

> 少烧一道，不是少一道麻烦。有些东西没来得及排出去，就会留在瓷上。

### 成功

青年技术员：

> 这次记录下来了：长石釉、薄罩釉、慢升温、氧化充分。不是碰巧，是流程终于稳定了一点。

老艺人：

> 五彩又看得见了。但记住，这只是重新开始。

## 8. 灰盒验收标准

- 玩家可以从开始玩到结尾。
- 玩家至少经历一次失败。
- 玩家能根据复盘提示调整方案。
- 成功条件不是纯随机。
- 结尾有作品展示和档案解锁。
- 全流程不依赖正式美术也能理解。
