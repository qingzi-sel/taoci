# 醴陵釉下五彩烧制工艺：游戏设计用考证稿

> 本文用于游戏设计，不作为工厂生产操作手册。  
> 醴陵釉下五彩的实际流程会因年代、器型、生产单位和装饰方法而变化。游戏中可以表现“三烧改二烧”的技术进步；本项目已经确认，采用二烧制的醴陵釉下五彩均在施有固定长石釉配方的素坯上进行彩绘。

## 一、可确定的基本原理

醴陵釉下五彩属于釉下彩的一种。它的核心特征是：

1. 使用能够承受高温的彩料，在坯体或经过前段处理的坯件上进行彩绘。
2. 彩绘完成后，在画面外覆盖透明釉。
3. 最终经过高温烧成，使坯体瓷化、釉层玻化并让彩料呈现烧成后的颜色。
4. 成品的纹样位于透明釉层之下，因此表面通常平滑，彩绘也较耐磨、不易褪色。

需要注意：

- 不宜说“颜料与釉层完全融合”。更准确的说法是：**高温烧成后，透明釉层覆盖并保护釉下彩绘，彩料在高温中完成呈色。**
- 不宜说“永不褪色”。釉下彩的耐磨和耐褪色性能通常优于釉上彩，但仍会受到烧成质量、材料和长期使用环境影响。
- 不宜固定使用某个烧成温度。不同胎、釉、色料和窑炉所需的烧成制度不同，游戏中使用“低温素烧”“高温烧成”即可。

## 二、传统三烧工艺的游戏用描述

根据现有资料，醴陵釉下五彩早期使用过包含三次入窑的工艺路线。为了避免对具体配方和操作作未经证实的断言，游戏中可简化为：

### 1. 第一次烧成：低温素烧

成型并干燥后的坯件先进行低温素烧，使坯件获得足够强度，便于后续彩绘和搬运。

### 2. 双勾与分水填彩

彩绘者先用特制的油性墨线勾出纹样边界，再在边界内使用釉下彩料进行分水填彩，通过料水浓淡、色料搭配和操作手法形成色彩层次。

这里原文中的“施墨”容易产生误解，建议统一改为：

> **分水填彩**或**施彩**

墨线主要承担辅助构图和阻隔料水的作用，不等同于最终留在画面上的黑色轮廓。墨线中的油性介质会形成临时的疏水边界，使水性彩料不易越界。

### 3. 第二次烧成：低温烘烧

彩绘后的坯件再次低温入窑，使墨线中的油、胶等有机辅助材料充分氧化烧失，并使画面适合后续施釉操作。这一步可以降低有机物在后续釉烧中造成发黑、气泡或针孔等问题的风险。

这一工序可称为“二次低温焙烧”或“低温脱脂焙烧”。公开整理资料对准确行业名称和温度范围的表述并不完全一致，游戏内可表现为“烧去油性墨线等有机辅助材料”，不宜加入具体温度数值。

### 4. 施透明釉

在彩绘表面覆盖透明釉。釉层需要尽量均匀；过厚、过薄或覆盖不均都可能影响最终呈色和表面质量。

### 5. 第三次烧成：高温烧成

器物最终高温烧成。坯体瓷化，透明釉形成玻璃质釉面，釉下彩料完成呈色。烧成后，彩绘位于釉层之下，画面与器物形成统一的视觉效果。

### 三烧工艺的简化流程

> **成型与干燥 → 低温素烧 → 墨线双勾 → 分水填彩 → 低温烘烧 → 施透明釉 → 高温烧成 → 选瓷**

## 三、改进后二烧工艺的游戏用描述

现有项目资料显示，后来形成了将三次入窑减少为两次的工艺路线。其关键并非简单删除一次烧成，而是重新安排前段施釉、素烧、彩绘和罩釉之间的关系。

### 1. 施长石釉并进行第一次低温烧成

坯件先施长石釉，再进行低温烧成，使坯件获得强度，并形成用于后续釉下彩绘的长石釉素坯表面。

传统旧工艺使用石灰青釉，烧成后的底色容易偏青、偏暗。二烧工艺改用固定配方的长石质白釉，并在最终烧成后形成透明釉层，使釉面更白、更透，显著提升釉下色彩的明亮度和洁净感。

游戏中应明确使用“长石釉”这一名称，不再泛称为“底釉”。

### 2. 双勾、分水填彩并覆盖薄层透明釉

彩绘者在第一次低温烧成后的坯件上完成双勾与分水填彩，再以喷釉等方式覆盖一层较薄、较均匀的透明釉。

“喷薄釉”适合在游戏中表现为一个重要操作，但不应暗示喷釉是所有作品唯一的罩釉方法。

### 3. 第二次高温烧成

器物经过最终高温烧成，坯体、釉层和彩料共同完成烧成变化，彩绘在透明釉层下呈现。

### 二烧工艺的简化流程

> **成型与干燥 → 施长石釉 → 第一次低温烧成，形成长石釉素坯 → 墨线双勾与分水填彩 → 覆盖薄层透明釉 → 第二次高温烧成 → 选瓷**

## 四、三烧改二烧应如何理解

游戏中可以将“三烧改二烧”表现为重要技术突破，但应避免以下误解：

1. **不是简单跳过第二次烧成。** 工序减少的同时，施釉位置、彩绘表面和罩釉方式也发生了调整。
2. **不是二烧一定比三烧品质更高。** 二烧的主要优势是缩短周期、减少能耗和人工，并有利于提高生产效率；成品质量仍取决于整体工艺控制。
3. **不是研究成功后就完全没有失败。** 二烧仍需要处理胎、釉、彩料和高温烧成之间的适配问题。
4. **不能把三烧表现成“落后而无价值”。** 三烧是历史发展阶段中的成熟工艺，也为后续改进积累了经验。

游戏可以让玩家通过试片与生产记录逐步解决：

- 长石釉固定配方是否稳定
- 长石釉素坯是否达到适合彩绘的状态
- 彩绘表面对分水效果的影响
- 覆盖釉层是否均匀
- 高温烧成后的呈色、变形和釉面质量
- 减少一次入窑后，成本、周期与成品率如何变化

三烧改二烧最适合在游戏中表现的核心难题是：

> **取消独立的低温脱脂焙烧后，如何让油性墨线等有机物在透明釉完全熔融封闭之前充分排出，同时保持画面、釉面和呈色质量。**

玩家需要共同研发并稳定长石釉固定配方，同时调整罩釉厚度、施釉方式和烧成过程，而不是简单解锁“减少一次烧成”的科技按钮。

## 五、双勾分水的准确游戏表达

双勾分水是醴陵釉下五彩具有代表性的装饰技法之一，适合成为核心交互。

### 双勾

“双勾”是用线条勾勒纹样边界和内部结构的画法，其核心是建立能够控制分水的区域，并不必然指两条平行墨线。

- 花瓣、叶片等纹样可以由外轮廓线与内部结构线共同划分分水区域。
- 装饰带可能实际使用两条近似平行线围出填彩区域。
- 枝蔓等不适合封闭分水的纹样，可能采用其他彩绘手法。

用于辅助分水的墨线会在后续烧成中烧失，烧成后可形成具有装饰作用的留白边界。

游戏中应表现为：

- 墨线是制作过程中的辅助线。
- 墨线帮助划分填彩区域、控制料水。
- 最终作品不应保留与烧前完全相同的黑色墨线。

留白并不是墨线“变成白色”，而是油性墨线曾经占据的位置没有被水性彩料覆盖；墨线烧失后，该区域透出胎釉底色，形成负形留白。游戏中可以把这一过程作为开窑后的视觉揭晓。

不建议宣称这种空白线“只有醴陵独有”。更稳妥的表述是：

> **烧失墨线形成的留白效果，是醴陵釉下五彩具有辨识度的装饰特征。**

### 分水

“分水”不是普通的平涂。彩绘者在轮廓范围内施加料水，通过料水浓淡、用量、流动和衔接形成色彩层次。

游戏中可以突出以下特点：

- 分水笔主要用于引导和控制料水，不应表现为像普通画笔一样反复涂抹坯面。
- 笔尖可以轻触色水或坯面附近以导引流动，但笔腹不应重压、拖擦彩绘表面。
- 料水浓淡和用量会影响烧成后的深浅与均匀程度。
- 操作需要考虑坯件倾斜方向、边界、积水、收水、衔接和干燥状态。
- 罩染、积水、接色、提色等名称可作为进阶技能，但其准确操作方式应在采访工艺师后再设计。

不宜宣称分水能够“调配出所有色系”。更准确的说法是：

> **随着耐高温色料的研发，醴陵釉下五彩的可用颜色和层次逐渐丰富，但具体颜色仍受色料配方与高温烧成条件限制。**

## 六、罩釉方式的游戏表达

不同器型、部位和生产目的可以采用不同施釉方式：

- **喷釉**：便于控制画面上的釉层厚度，适合艺术瓷、不规则器型、大件或需要保护精细彩绘的表面。
- **浸釉**：效率较高，适合杯、碗、盘等器型规整的批量产品，但操作不当可能冲刷未固定的彩绘或在局部积釉。
- **荡釉**：主要用于瓶、壶、杯等空心器物的内壁。
- **刷釉**：可用于局部补釉或特殊效果，不宜作为游戏中所有产品的默认罩釉方式。

游戏中可以让玩家根据器型和彩绘选择施釉方式，但不需要模拟全部操作细节。

## 七、烧成风险如何在游戏中真实表现

釉下彩料和透明釉需要经历高温烧成。最终呈色和质量会受到材料、釉层、窑炉气氛、温度过程、器物位置等因素影响，因此很多问题要到开窑后才能最终确认。

游戏中可以表现：

- 烧前色与烧后色存在差异。
- 试片和色标能降低不确定性，但不能完全消除风险。
- 高温烧成可能出现色差、变形、开裂、釉面不均、针孔、气泡等问题。
- 玩家可以根据失败结果追查原因并改进，而不是面对完全随机的失败。

不建议直接断言“釉下五彩成品率一定很低、成本一定很高”。不同年代、产品和生产条件下的成品率差异很大。游戏内可改为：

> **釉下五彩对彩料与高温烧成的配合要求较高；在工艺尚未稳定时，失败会显著增加成本。**

## 八、可直接用于游戏的最终流程

### 教学阶段：传统三烧

> **低温素烧 → 墨线双勾 → 分水填彩 → 低温烘烧 → 施透明釉 → 高温烧成 → 开窑选瓷**

玩家应理解：

- 为什么需要辅助墨线
- 分水与普通填色的区别
- 为什么要罩透明釉
- 三次入窑带来的时间与成本压力

### 研发阶段：改进二烧

> **施长石釉 → 第一次低温烧成，形成长石釉素坯 → 墨线双勾与分水填彩 → 覆盖薄层透明釉 → 第二次高温烧成 → 开窑选瓷**

玩家应理解：

- 二烧不是单纯删除一道工序
- 新流程需要研发稳定的长石釉配方，并解决长石釉素坯彩绘、罩釉和高温烧成之间的配合
- 技术进步来自试验记录和多人协作

## 九、已提炼结论与仍需核实内容

### 已可用于游戏设计的结论

1. 三烧工艺的第二次低温烧成主要用于烧去油性墨线等有机辅助材料。
2. 本项目已确认，采用二烧制的醴陵釉下五彩均在使用固定长石釉配方烧成的素坯上彩绘。
3. 从石灰青釉转向长石质白釉，使釉面更白、更透，显著提升釉下颜色的明亮度和洁净感。
4. 留白线来自油性墨线阻水占位、烧失后透出胎釉底色的负形留白效果。
5. 双勾不必然指两条平行线，其核心是通过轮廓线和结构线控制分水区域。
6. 分水动作以轻触导流为主，不能表现为重压、来回涂抹。
7. 喷釉、浸釉和荡釉适用于不同器型与部位，喷釉不是唯一罩釉方式。
8. 三烧改二烧的核心难题，是在取消独立脱脂焙烧后仍保证有机物排出、釉面质量和彩料呈色。

### 仍需原始史料或工艺师核实

1. 各次烧成的准确温度、保温时间和窑炉气氛。
2. “二次低温素烧”“二次低温焙烧”“脱脂焙烧”等名称在醴陵工艺实践中的标准用法。
3. 墨线的具体配方，以及不同年代、工坊是否使用相同材料。
4. 长石釉固定配方的准确组成、编号，以及历史资料中对其研发过程的记录。
5. 三烧改二烧的准确时间、参与人员和各自贡献。
6. 罩染、积水、接色、提色等进阶分水技法的准确动作。

在这些问题得到确认前，游戏可以采用本文的简化流程，但不要出现具体温度、配方比例、真实人物功绩归属或“唯一工艺”等断言。

### 本次修改的主要结论

- 保留“三烧改二烧”作为游戏中的核心技术进步，但将两套流程标明为特定工艺路线，而非唯一标准。
- 将“施墨”改为“分水填彩／施彩”，避免误解为只用墨色作画。
- 将“颜料与釉层融合”改为“透明釉覆盖并保护彩绘，彩料在高温中完成呈色”。
- 将“永不褪色”“独一无二”“涵盖所有色系”等绝对说法改为可验证、不会误导的表述。
- 保留墨线烧失、留白效果和分水技法作为醴陵釉下五彩的重要游戏特色，但把动作与材料细节列入实地核实清单。
- 根据七项问题回答，补充油性墨线阻水与负形留白原理、分水笔导流动作、不同罩釉方式及三烧改二烧的核心技术难题。
